# Tutorial: First-time developer

> **Navigation:** [Tutorials INDEX](INDEX.md)

1. Follow [quick-start](../getting-started/quick-start.md)
2. Read [system overview](../architecture/system-overview.md)
3. Open Swagger: http://localhost:3001/api/docs
4. Run `node scripts/verify.mjs` with API up
5. Explore [API INDEX](../api/INDEX.md) for module you’ll change

**Checkpoint:** You can register via API or UI and see plants in Prisma Studio (`npm run db:studio`).
