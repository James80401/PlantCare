# Tutorial: First-time user

> **Navigation:** [Tutorials INDEX](INDEX.md) · [User guide: auth](../user-guide/landing-and-auth.md)

1. Open http://localhost:5173
2. **Register** (or **Login** if account exists)
3. If email verification enabled — click link in inbox → `/verify-email/:token`
4. Land on **Garden** (`/garden`) — dashboard of your plants
5. **Add plant** → `/garden/plants/new`

Next: [Adding a plant](adding-a-plant.md)
