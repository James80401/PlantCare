# Plant Buddy — Customization System

## Overview

Customization is one of the primary reward loops. Players earn Dewdrops through tasks and journeys and spend them in the Shop to personalize their buddy and terrarium. All customization is purely cosmetic — it never affects gameplay progression.

---

## Customization Categories

There are 8 equippable slots on the buddy itself, plus the terrarium environment.

| Slot | Description | Max Items Planned |
|---|---|---|
| Hat | Worn on top of buddy | 40+ |
| Top | Body covering/outfit | 30+ |
| Shoes / Roots | Base covering | 20+ |
| Glasses | Eye accessory | 15+ |
| Held Item | Item buddy carries | 25+ |
| Pot Skin | The planter the buddy sits in | 30+ |
| Body Color | Base color variant | 6 per species |
| Body Pattern | Overlay pattern on body | 8 patterns |
| Companion | Small creature following buddy | 8 companions |
| Wings / Extras | Back slot — wings, auras | 10+ |

## Database: Shop & Inventory Models

```prisma
model ShopItem {
  id              String        @id @default(cuid())
  name            String
  description     String
  category        ItemCategory
  tier            Int           // 1–4
  cost            Int           // in Dewdrops (0 = free, -1 = not purchasable)
  isSeasonalOnly  Boolean       @default(false)
  seasonMonths    Int[]         // months available, empty = always
  speciesLocked   String?       // species id if species-exclusive, null = all
  unlockType      ItemUnlockType @default(PURCHASE)
  unlockCondition String?       // description of how to unlock if not purchased
  imageKey        String        // reference to asset in CDN
  isActive        Boolean       @default(true)
  createdAt       DateTime      @default(now())

  inventory       BuddyInventory[]
}

model BuddyInventory {
  id          String    @id @default(cuid())
  buddyId     String
  buddy       Buddy     @relation(fields: [buddyId], references: [id])
  itemId      String
  item        ShopItem  @relation(fields: [itemId], references: [id])
  acquiredAt  DateTime  @default(now())
  acquireMethod String  // "purchase", "quest", "journey", "event", "default"

  @@unique([buddyId, itemId])
}

enum ItemCategory {
  HAT
  TOP
  SHOES
  GLASSES
  HELD_ITEM
  POT_SKIN
  BODY_COLOR
  BODY_PATTERN
  COMPANION
  WINGS
  BACKGROUND
  FURNITURE
}

enum ItemUnlockType {
  PURCHASE       // Dewdrops
  QUEST_REWARD
  JOURNEY_DROP
  ACHIEVEMENT
  SEASONAL_EVENT
  SPECIES_UNLOCK
  DEFAULT_ITEM
}
```

## Daily Shop Rotation

The shop has two sections:

**1. Permanent Catalog** — always available, browsable by category

**2. Daily Rotation** — 4 items rotate every 24 hours at midnight UTC. These are items from Tier 2–3 at a slight discount (10–15% off). Creates daily engagement reason to check the shop. The rotation is seeded by date so all users see the same daily rotation.
