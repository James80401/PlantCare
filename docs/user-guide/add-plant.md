# User guide: Add plant

> **Navigation:** [User guide INDEX](INDEX.md) · [Web: add plant](../web/pages/add-plant.md) · [Tutorial: adding a plant](../tutorials/adding-a-plant.md)

**Route:** `/garden/plants/new`

Reach from the garden dashboard **+ Add plant** button or layout navigation.

## Form fields

- **Species search** — type at least 2 characters; pick from catalog results
- **Identify from photo** (optional) — suggests species when PlantNet is configured
- **Nickname** (optional)
- **Where it grows** — preset locations (Living Room, Garden, Outdoor garden, …)
- **Pot size** — small, medium, large (affects watering interval)
- **Date planted** (optional)
- **Photo** (optional) — upload for the plant card

## After save

You are taken to the plant profile. The backend schedules ~90 days of care tasks (water, prune, and premium tasks per plan rules).

## Tips

- Use **Garden** or **Outdoor garden** for outdoor plants — see [outdoor vs indoor](../tutorials/outdoor-vs-indoor-plants.md)
- Plant `notes` at create are **API-only** today (not on this form) — see [feature availability](../reference/feature-availability.md)
