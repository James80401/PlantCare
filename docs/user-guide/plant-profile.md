# User guide: Plant profile

> **Navigation:** [User guide INDEX](INDEX.md) - [Web: plant profile](../web/pages/plant-profile.md)

**Route:** `/garden/plants/:id`

| Section | Purpose |
|---------|---------|
| Header | Photo, nickname, species, current status, location, quick care summary, and primary actions |
| Overview | What matters now, next care, health state, recent care, species facts, recommendations, details, and location |
| Care | Beginner-first care guidance, environment context, safety notes, and Dr. Plant links for each topic |
| Tasks | Pending care tasks, instructions, complete/skip/snooze actions, and recent care history |
| Journal | Notes, photos, Plant Life check-ins, measurements, and timeline history |
| Health | Dr. Plant chat, diagnosis history, recovery follow-up, and active/resolved status |

**Change location** refreshes pending tasks for the new growing spot - [tutorial](../tutorials/changing-plant-location.md).
